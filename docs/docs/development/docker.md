# Docker

Use the project `Dockerfile` or `docker-compose` files to run LaraPaper in a container.

Production compose: [docker/prod/docker-compose.yml](https://github.com/usetrmnl/larapaper/blob/main/docker/prod/docker-compose.yml).

For interactive development with VS Code, prefer the [Devcontainer](/development/devcontainer) workflow.
