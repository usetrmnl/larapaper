# Requirements

## Runtime

- PHP **>= 8.4**
- PHP extension **imagick**
- **Puppeteer** for screen rendering — see [Browsershot requirements](https://spatie.be/docs/browsershot/v4/requirements)

## Development

- Composer
- Node.js (for frontend assets: `npm run build`)
- SQLite (default), or MySQL / PostgreSQL
